### 搭建
`git clone https://github.com/Virgo9/react-app.git`
`cd react-app`

`yarn install`
`yarn start`


### 目录结构
```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    index.css
    index.js
    logo.svg
```


### 创建新应用
`npm install -g create-react-app`
`create-react-app my-app`

`cd my-app`
`npm start`
